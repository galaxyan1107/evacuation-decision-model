## 主要发现

### 1. 攻击强度正在下降但未归零

弹道导弹从第1天的137枚降至第5天的个位数，但第7天出现小幅反弹（7→9），打破单调递减趋势。无人机维持在约112-130架/天的稳定水平，受限于产能上限。每日威胁总量较峰值下降65%。

![每日新增攻击分类 — 阿联酋国防部数据](../assets/daily_attacks.png)

### 2. 弹道导弹威胁呈指数衰减；无人机持续存在

30天预测显示，在贝叶斯衰减率（β=0.25/天）下，弹道导弹到第15天接近零，而无人机在整个预测期内维持约130架/天。无人机成为主要的残余威胁载体。

![攻击预测 — 30天展望](../assets/attack_forecast.png)

### 3. 基线情景占主导；升级风险下降

基于50K蒙特卡罗模拟的贝叶斯ABC分析，到第30天，基线情景（持续但不升级的冲突）概率为75.4%。随着发射装置消耗累积，停火概率升至12.8%。升级概率从第6天的31.4%降至第30天的11.7%。全面战争概率可忽略（<1%）。

![后验情景概率随时间演变](../assets/bayesian_posteriors.png)

### 4. 系统处于亚稳态 — 存在厚尾级联风险

HAM反馈增益λ中位数为0.739（低于级联阈值1.0），表明稳定力量占主导。然而，12.2%的蒙特卡罗模拟显示λ > 1.0，由厚尾风险驱动：代理人激活（胡塞/真主党，P=6%）、霍尔木兹海峡封锁（P=4%）和拦截能力退化。第95百分位λ达到1.52。

![HAM反馈增益λ分布 — 50K蒙特卡罗](../assets/lambda_distribution.png)

### 5. 两个航母打击群提供强大稳定效应

美国在中东集结了自2003年以来最大规模的军事力量：CVN-72亚伯拉罕·林肯号部署在阿拉伯海，CVN-78杰拉尔德·R·福特号部署在东地中海，另有6艘独立的宙斯盾反导驱逐舰。该兵力态势对λ产生约-0.16的稳定效应，并降低了代理人激活和霍尔木兹海峡封锁的概率。

![海军兵力态势 — 史诗之怒行动](../assets/naval_posture.png)

### 6. 人口外流遵循明确层级

过境旅客和游客几乎立即离开（第20天前>90%）。短期外籍人士随后离开，第25天前约75%离境。长期家庭——最大群体（450万人）——在第8-10天左右开始离开，因为持续压力和社会传染效应累积。高净值/商务人士和本国公民最为坚守，本国公民在第30天时离境率仍低于5%。

![按代理类型划分的人口离境曲线](../assets/population_flight.png)

### 7. 撤离风险评分早期达峰、快速衰减

综合风险评分在第1天约为100（立即撤离），第3天降至80以下，第6天降至60以下（"黄金窗口"关闭）。第10天达到约45（可接受但有风险），第30天稳定在27左右。早期主要驱动因素是机场排队瓶颈（运力为正常水平的0-15%），其次是攻击强度和级联尾部风险。

![综合撤离风险评分](../assets/risk_score.png)

## 验证摘要

将第7天来自 `@modgovae`、Polymarket和GCAA的实时数据与模型预测进行对比：

| 检查项 | 模型 | 观测值 | 结果 |
|--------|------|--------|------|
| 弹道导弹单调递减 | 是 | 7→9（打破） | **偏离** |
| 拦截率 > 90% | 93.2% | 92.7% | **吻合** |
| 无人机率稳定 | ~130/天 | 112/天 | **接近** |
| 无新型武器 | 无 | 无 | **吻合** |
| 停火概率 (3月31日) | 84% | 63% (Polymarket) | **偏离** |
| 机场恢复 | 25% | ~25% (GCAA) | **吻合** |
| 无人机库存 > 30% | 40.8% | ~40% | **接近** |
| 第30天离境人数 | 504万 | 555万（报告） | **偏离** |

**所有偏离均加强了撤离建议** — 模型略微低估了残余威胁（弹道导弹反弹、Polymarket较低的停火概率）和离境人数。

## 模型架构

```
国防部数据 (@modgovae)
    → 1. 态势评估（趋势分析、衰减率）
    → 2. 贝叶斯ABC预测（50K蒙特卡罗、4种情景、软核函数）
    → 3. HAM稳定性分析（λ反馈增益、级联概率）
    → 4. 人口外流模型（6种代理类型、Brock-Hommes 1998）
    → 5. 撤离风险评分（综合0-100分、离境窗口）
```
