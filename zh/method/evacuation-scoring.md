# 5. 撤离风险评分

> 🌐 [English](../../method/evacuation-scoring.md) | **中文**


最终模块生成**综合撤离风险评分（0-100）**，将所有模型输出综合为单一可操作指标。

## 综合评分公式

```
S(d) = min(衰减包络(d),
        w_atk · ATK(d) + w_casc · CASC(d) + w_queue · QUEUE(d)
      + w_oman · OMAN(d) + w_flight · FLIGHT(d))
```

### 分量权重（默认）

| 分量 | 权重 | 描述 |
|------|------|------|
| 攻击强度 | 0.25 | 归一化每日攻击威胁水平 |
| 级联概率 | 0.20 | HAM λ > 1 累计概率 |
| 机场排队 | 0.20 | 归一化离境排队压力 |
| 阿曼溢出效应 | 0.15 | 区域传染风险 |
| 外流紧迫性 | 0.05 | 对数时间压力 |

### 衰减包络

综合评分受指数衰减包络约束，校准以匹配报告的风险轨迹：

```
衰减包络(d) = 95 · exp(-0.13 · d) + 25
```

这确保评分遵循报告的95 → 27轨迹（10天内），反映初始攻击强度消退和机场运力恢复带来的紧迫性递减。

## 建议等级

| 评分区间 | 建议 |
|---------|------|
| ≥ 80 | **立即撤离** |
| ≥ 60 | **撤离 — 黄金窗口** |
| ≥ 40 | **可接受 — 排队增加中** |
| ≥ 30 | **偏晚 — 有风险** |
| < 30 | **低风险 — 但需持续监控** |
