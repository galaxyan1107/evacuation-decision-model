# 3. HAM稳定性分析

> 🌐 [English](../../method/ham-stability.md) | **中文**


异质代理模型稳定性模块评估冲突系统处于**稳定均衡**、**亚稳态**（对小冲击有弹性）还是接近**级联临界点**。

## 核心指标：反馈增益 λ

核心诊断指标是反馈增益λ。当 λ > 1.0 时，正反馈回路占主导，系统进入升级级联。

```
λ > 1.0  →  级联（正反馈占主导）
λ < 1.0  →  稳定/亚稳态（负反馈占主导）
```

## 级联阈值指标

监测五项二元阈值指标：

| 指标 | 数值 | 阈值 | 状态 |
|------|------|------|------|
| 发射装置消耗率 | 85.7% | > 85% | **已突破** |
| 无人机库存剩余 | 40.8% | < 30% | 正常 |
| 拦截率 | 92.7% | < 90% | 正常 |
| 每日伤亡 | 16.4 | > 10 | **已突破** |
| 新型武器 | 无 | 有 | 正常 |

**判定规则：** ≥3项突破 = 不稳定，≥1项 = 亚稳态，0项 = 稳定

当前状态：**亚稳态**（5项中2项突破）

## 反馈增益 λ 分解

反馈增益通过50K蒙特卡罗模拟计算为六个随机效应之和：

```
λ = 1.0 + λ_发射装置 + λ_无人机 + λ_拦截 + λ_代理人 + λ_霍尔木兹 + λ_武器
```

### 稳定效应（负反馈）

**发射装置消耗** — 约350台TEL中约300台被联军摧毁：
```
λ_发射装置 = -0.55 · 消耗率 · (1 + N(0, 0.12))
```
这是主要的稳定力量，将λ显著拉低至1.0以下。

### 不稳定效应（正反馈）

**无人机持续性** — 产能限制在约130架/天维持压力：
```
λ_无人机 = +0.25 · (1 - 库存剩余率) · (1 + N(0, 0.2))
```

**拦截能力退化** — 防御系统退化的厚尾风险：
```
λ_拦截 = +0.15 · Exp(0.03) / (1 - 拦截率 + 0.01)
```

**代理人激活** — 胡塞武装/真主党加入（P=6%，高影响）：
```
λ_代理人 = +0.8 · Bernoulli(0.06) · U(0.5, 2.0)
```

**霍尔木兹海峡封锁** — 海峡封锁风险（P=4%）：
```
λ_霍尔木兹 = +0.6 · Bernoulli(0.04) · U(0.6, 1.5)
```

**新武器引入** — 新型能力出现（P=3%）：
```
λ_武器 = +0.4 · Bernoulli(0.03) · U(0.5, 1.5)
```

## 结果

| 指标 | 数值 |
|------|------|
| λ 中位数 | 0.729 |
| λ 第95百分位 | ~1.35 |
| 每日级联概率 | P(λ > 1) 每次模拟 |
| 14天累计级联概率 | 1 - (1 - P_日)^14 |

**解读：** 系统处于亚稳态 — 发射装置消耗提供强大的稳定力量，但代理人激活或霍尔木兹海峡封锁的厚尾风险在14天展望期内创造了不可忽视的级联概率。

> **更新（3月7日）：** 随着第3个航母打击群（CVN-77乔治·H·W·布什号）即将部署，海军威慑效应从-0.16增加至-0.24。代理人激活概率降至4%，霍尔木兹封锁风险降至2%，总体P(λ > 1)从约12.2%降至估计约8-9%。
