## 弹道导弹衰减模型

弹道导弹发射率遵循指数衰减模式：

```
BM(t) = BM₀ · exp(-β · t) + εₜ
```

其中：
- `β` = 每日衰减率
- `BM₀` = 137（第1天观测到的首轮齐射）
- `εₜ` = 噪声项

**关键差异：** 报告使用贝叶斯后验估计 β = 0.25/天，而简单OLS对数线性回归得出 β = 0.608/天。OLS因第1天首轮齐射（137枚弹道导弹）后的急剧下降而被抬高。贝叶斯方法通过基于历史冲突衰减模式的先验信息进行正则化。
